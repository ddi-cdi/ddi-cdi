CollectionsPattern¶

Fully qualified package name: DDICDIModels::DesignPatterns::CollectionsPattern

Template classes describing a group or collection of elements and their interrelationships. These classes support description of the way in which individual units of meaning, their representations, and other items of metadata can be assembled into groups with common characteristics and purpose.
