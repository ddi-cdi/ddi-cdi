DCTerms¶

Fully qualified package name: DDICDIModels::Context::DCTerms

This package contains classes taken from DCMI Metadata Terms (https://www.dublincore.org/specifications/dublin-core/dcmi-terms/#section-1) which are used in the DDI-CDI model. These classes represent the properties coming from both the elements and terms namespaces, represented as classes for inclusion in the DDI-CDI model.
