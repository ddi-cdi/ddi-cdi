contributor
***********

**Fully qualified datatype name**: DDICDIModels::Context::DCTerms::contributor

Definition
==========
An entity responsible for making contributions to the resource. The guidelines for using names of persons or organizations as creators also apply to contributors. Typically, the name of a Contributor should be used to indicate the entity.
