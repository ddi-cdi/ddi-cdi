AgentStructure
**************

**Fully qualified class name**: DDICDIModels::DDICDILibrary::Classes::Agents:::index:`AgentStructure`


Definition 
============ 
Defines the relationships between agents in a collection for a specified period and purpose.  

Examples 
========== 
:ref:`DDICDIModels::DDICDILibrary::Classes::Agents::Individual` employed by an organization. A unit or project (organization) within another organization. :ref:`DDICDIModels::DDICDILibrary::Classes::Agents::Individual` supervised by another individual.  

Explanatory notes 
=================== 
Can describe relations between agents rather than roles within a project or in relationship to a product. Roles are defined by the parent class and relationship name that uses an agent as a target.
