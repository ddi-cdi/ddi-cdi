UCMIS-M2T
=========

The purpose of the UML Class Model Interoperable Subset - Model to Text (UCMIS-M2T) software
is to generate documentation and syntax representations (encodings) for a UML class model in a model-driven approach.
UCMIS-M2T can be used for any UML model that conforms to the :ref:`about_ucmis`.

UCMIS-M2T is developed for the specification
`DDI Cross Domain Integration (DDI-CDI) <https://ddialliance.org/Specification/ddi-cdi>`_.

Further details can be found at the corresponding Git repository https://bitbucket.org/wackerow/ucmis.m2t/.
